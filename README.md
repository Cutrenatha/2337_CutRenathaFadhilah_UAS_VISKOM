Proyek Fruit Freshness Detection merupakan aplikasi berbasis Computer Vision yang bertujuan untuk mengklasifikasikan tingkat kesegaran buah menggunakan citra digital. Sistem ini mampu mengidentifikasi kondisi buah segar (fresh) atau busuk (rotten) pada beberapa jenis buah, yaitu apel, pisang, dan jeruk, berdasarkan gambar yang diberikan pengguna.

Model dikembangkan menggunakan metode YOLO Classification dan diimplementasikan dalam aplikasi berbasis Python dengan antarmuka web menggunakan Gradio, sehingga mudah digunakan dan interaktif.
